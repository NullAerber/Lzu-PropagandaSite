传承

清末时期

1908年11月8日（清朝光绪三十四年十月十五日），甘肃法政馆将法政馆改为甘肃法政学堂。

1909年（宣统元年），改法政馆为甘肃法政学堂，划归甘肃提学使司。

1909年2月5日，将甘肃法政馆改为“甘肃官立法政学堂”，颁发“甘肃法政学堂关防”，性质为高等学堂。

民国时期

1913年6月，法政学堂后，改组法政学堂为“甘肃公立法政专门学校”，校舍定于兰州西关萃英门旧举院内，至公堂、观戏堂为教室。

1925年7月15日，奉冯玉祥令，甘肃省政府主席刘郁芬批准将甘肃省立五族学校，更名为甘肃省立中山学院。

1928年，甘肃省立中山学院扩建为兰州中山大学。

1945年，改为国立兰州大学。

建国之后

1949年，兰州解放。全国各地学校一律丢掉居首“国立”、“省立”等字样，国立兰州大学更为兰州大学。

1959年，原北京大学副校长、党委书记江隆基奉调入主兰州大学，成为校长。

改革开放

1996年，首批通过“211工程”部门预审，成为国家在“九五”期间重点建设的高校。

2001年，成为国家“985工程”重点支持建设高水平大学的高校之一。

2009年，兰大迎来百年校庆。
